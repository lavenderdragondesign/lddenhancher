How to Get Your Free Gemini API Key

The Image Describer feature requires a free Google Gemini API key. It's quick to set up and is stored only in your browser for privacy.

  1. Click this link to go to Google AI Studio: Get Your Free Gemini API Key.
  2. Click the button to Create API key. If prompted, you must select the option to Create API key in new project.
  3. A pop-up will appear with your new key. Click the Copy icon (it looks like two overlapping squares) next to the long string of characters.
  4. After installing the script, open it inside MyDesigns, go to the Settings tab, paste your key into the box, and click Save Key. That's it!
