Step 1: Create Your Gemini API Key

Go to Google AI Studio and click 'Create API key'. If prompted, choose 'Create API key in new project'.

Step 2: Copy the API Key

Click the Copy icon (looks like two overlapping squares) to copy your Gemini key.

Step 3: Install Tampermonkey

Install the Tampermonkey browser extension from the Chrome Web Store (works for Edge too).

Step 3a: Allow Permissions

When prompted, allow necessary permissions for the extension to function.

Step 4: Install the Enhancer Script

Click the green 'Install Script' button on this page to launch the Tampermonkey script installer.

Step 4a: Confirm Script Install

Click the 'Install' button in the top-left corner of the Tampermonkey tab.

Step 5: Open MyDesigns

Go to MyDesigns.io and look for the glowing Enhancer logo in the bottom-right corner.

Step 6: Paste Your Gemini API Key

Open the Enhancer > Settings tab, paste your API key, and click Save.
