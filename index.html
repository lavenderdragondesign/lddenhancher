Step 1: Create Your Gemini API Key

Go to Google AI Studio and click 'Create API key'. If prompted, choose 'Create API key in new project'.

Go to Google AI Studio

Step 2: Copy the API Key

Click the Copy icon (it looks like two overlapping squares) to copy your Gemini key to your clipboard.

Step 3: Install Tampermonkey

Visit tampermonkey.net or use the Chrome Web Store. Tampermonkey works on Chrome, Edge, Brave, and more.

Install Tampermonkey Extension

Step 3a: Allow Permissions

After clicking install, your browser may ask for permission to add the extension. Click 'Add Extension'.

Step 4: Install the Script

Back on the Enhancer page, click the big green 'Install Script' button to launch Tampermonkey's install window.

Click the green "Install Script" button

Step 4a: Confirm Script Installation

Tampermonkey will show the script code. Just click the 'Install' button in the top-right.

Step 5: Open MyDesigns and Launch Enhancer

Go to mydesigns.io/app/listings. Look for the glowing LavenderDragon circle in the bottom-right corner.

Go to MyDesigns.io Listings

Step 6: Paste Gemini API Key in Settings

Click the Settings tab in the Enhancer. Paste your copied Gemini key into the box and click Save.
